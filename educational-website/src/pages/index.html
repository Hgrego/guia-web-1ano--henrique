Fundamentos

HTML

Estrutura e semântica para seu conteúdo web.

<article>
  <h2>Título</h2>
  <p>Conteúdo</p>
</article>

Use para estruturar e organizar o conteúdo da página.

CSS

Estilização e layout para suas páginas.

.card {
  display: flex;
  padding: 1rem;
  border-radius: 8px;
}

Use para definir aparência e posicionamento dos elementos.

JavaScript

Interatividade e funcionalidades dinâmicas.

document.querySelector('.button')
  .addEventListener('click', () => {
    console.log('Clicado!');
});

Use para adicionar comportamento e interatividade.
